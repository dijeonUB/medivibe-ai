# ROADMAP — MediVibe AI 개발 로드맵

## 개발 철학 (Karpathy Guidelines 적용)

1. **단순하게 시작하라** — 채팅 UI 하나로 시작, 복잡도는 필요할 때만 추가
2. **직접 눈으로 확인하라** — 각 스프린트마다 브라우저에서 직접 검증
3. **과도한 추상화를 경계하라** — 현재 필요한 기능만 구현
